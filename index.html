Deployable multi-step static app

Use of Force Reporting System

This version organizes the form into sequential steps, adds conditional fields, standardizes key dropdowns, and submits directly to Supabase. It is designed to be much simpler to complete than a long single-page form.

Included

Multi-step flow, progress indicator, conditional fields, required submission metadata, confirmation review step, PDF export, email draft option, free-text comments, and file attachment input.

Important

The attachment field is included in the HTML flow, but file upload storage is not wired to Supabase Storage yet.

  • Run your Supabase SQL setup first.
